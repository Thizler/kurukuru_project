# 🍥 KuruKuru Project

**KuruKuru** คือเว็บไซต์ขายสินค้าออนไลน์ที่ได้รับแรงบันดาลใจจากประสบการณ์การใช้งานแบบ **TikTok**  
ผู้ใช้สามารถเลื่อนดูคลิปวิดีโอรีวิวสินค้าแบบต่อเนื่อง และสามารถกดซื้อสินค้าได้ทันทีจากหน้าเดียวกัน

---

## ✨ Features
- 📱 เลื่อนดูวิดีโอสินค้าในรูปแบบแนว TikTok
- 🎥 แสดงวิดีโอรีวิวสินค้า
- 🛒 ซื้อสินค้าได้ทันทีจากหน้าวิดีโอ
- 📦 แสดงข้อมูลสินค้า (ชื่อ, ราคา, รายละเอียด)
- 👤 ระบบผู้ใช้งาน (พื้นฐาน)
- 🧪 รองรับการทดสอบด้วย Jest

## 🚀 How to Run Project
1. Clone repository
```bash
git clone https://github.com/Thizler/kurukuru_project.git
```
เข้าไปที่โฟลเดอร์โปรเจกต์

```bash
cd kurukuru_project
```
ติดตั้ง dependencies

```bash
npm install
```
รันโปรเจกต์

```bash
npm start
```
จากนั้นเปิดเบราว์เซอร์ที่ http://localhost:3000

🧪 Run Tests
```bash
npm test
```
